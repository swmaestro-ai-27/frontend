# Audio Policy

## Current implementation

All background music and sound effects are synthesized at runtime with the Web
Audio API in `src/lib/audioDirector.ts`.

- No third-party audio recordings are bundled.
- No copyrighted composition or sample is copied.
- The generated noise and oscillator patterns are original project code.
- No audio attribution is required for the current implementation.

## Replacing synthesized sounds

Only add downloaded audio when its license is recorded in this document.
Preferred licenses:

1. CC0 / public domain
2. A custom asset created for this project with explicit reuse permission
3. CC BY, only when attribution and redistribution requirements are satisfied

Do not add assets marked "personal use", "non-commercial only", or with an
unclear source. This project being non-commercial does not remove copyright
requirements.

For every external asset, record:

- Original file name
- Creator
- Source URL
- License name and version
- Download date
- Local file path
- Any required attribution text
